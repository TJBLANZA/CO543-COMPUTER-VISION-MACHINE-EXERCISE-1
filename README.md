# CO543-COMPUTER-VISION-MACHINE-EXERCISE-1

RED LIGHT GREEN LIGHT (RLGL)

This project implements a Red Light Green Light (RLGL) game using computer vision and webcam input. The goal of the system is to detect player movement using real-time image processing and enforce the rules of the RLGL game. In the game, players must move during GREEN light and remain completely still during RED light.

The system captures frames from a webcam and analyzes the difference between consecutive frames to determine movement. A state machine is used to control the game flow, switching between states such as GREEN, RED, WARNING, and DEAD. This exercise demonstrates how motion detection and real-time computer vision can be used to create an interactive application.
